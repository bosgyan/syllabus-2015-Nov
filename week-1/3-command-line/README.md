## Please install git on your system:
  - Windows:
    - http://cmder.net/ or https://git-scm.com/download/win
  - Linux
    - `sudo apt-get install git` (Ubuntu)
    - `sudo yum install git` (Fedora)
  - Mac
    - http://git-scm.com/download/mac or `brew install git`
